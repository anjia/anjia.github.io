快捷键

编辑

基本编辑

新增一行，在当前行的上面 Ctrl + Enter 【cmd＋Enter】
新增一行，在当前行的下面 Ctrl + Enter + Shift 【cmd＋Enter＋Shift】

逐词移动 Ctrl + 左右箭头 【Option＋左右箭头】
逐词选择 Ctrl＋左右箭头 + Shift 【Option＋左右箭头+Shift】

移动当前区域 Ctrl+ 上下箭头
移动当前行 Ctrl＋上下箭头 + Shift

选择

多重选择，同时编辑 Ctrl+D 【Cmd＋D】：按一次 再按一次 再按一次
My name is KikiAn.
KikiAn is me.
KikiAn is a good girl.

对一片区域，同时编辑（首先全部选中）
1.Lines 打散－多行编辑 Ctrl+Shift+L 【Cmd+Shift+L】
2.Join 多行合并 Ctrl+J 【Cmd＋J】
var array = [‘1’, ‘2’, ‘3’, ‘4’, ‘5’, ‘6’];
var value = {
‘A-anjia Hey Guys HaHa’’B-anjia Hey Guys HaHa’’C-anjia Hey Guys HaHa’’D-anjia Hey Guys HaHa’};

查找和替换

Ctrl+F 【Cmd+F】
Ctrl+F+Shift 【Cmd+F+Shift】 多文件查找

Ctrl＋H 标准替换

关键字查找：输入key后Enter，此时
Enter 下一个关键字
Enter+Shift 上一个关键字
Enter＋Alt（Enter＋Option）选中出现的所有位置

自动完成

提示后，按 Tab 会自动补全

跳转 Jumping

Sublime Text 把 Ctrl＋P 称之为 “Go To Anything” ［好多内涵］

跳转到文件

ctrl+P 【cmd+P】 后，输入文件名
嗯～字符串 模糊匹配 Fuzzy String Matching

跳转到符号|定义处|某行

代码编辑器-对代码符号进行索引

  1. ctrl+R - 列出当前文件中的符号（类名 函数名，但无法深入到变量）
    对于 Markdown Ctrl+R 会列出大纲
  2. F12 快速跳转到当前光标所在符号的定义处。 Jump to Definition
  3. 跳转到某行，Ctrl＋G

组合跳转

编码风格：缩进|留白|格式化

良好的实践，规范化自己的代码（通过设置来统一搞定）缩进 留白

  1. Sublime Text 基本的手动格式化操作：
    向左缩进 Ctrl+[ 【Cmd＋[】
    向右缩进 Ctrl+] 【Cmd＋]】
    以当前缩进粘贴代码 Cmd+Shift+V
  2. 插件：自动缩进和智能对齐
    格式化 HTML：HTMLBeautify
    智能对齐：Alignment
  3. 括号：
    起始括号间切换 Ctrl+M
    快速选择括号间的内容 Ctrl+M+Shift

文件夹

支持以文件夹作为单位进行编辑
Ctrl + K
Ctrl + B (Cmd+B)
