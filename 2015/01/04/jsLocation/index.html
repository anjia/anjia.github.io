Location 对象

Location 对象是 Window 对象的一个部分，可通过 window.location 属性来访问。
Location 对象包含有关当前 URL 的信息。

Location 对象的属性

以下属性均是字符串，且可读可写

  1. protocal：协议
  2. hotsname：主机名称
  3. port：端口号
  4. host：主机名称+端口号
  5. pathname：路径
  6. search：查询部分，从?开始
  7. hash：锚部分，从#开始
  8. href：完整的 URL

Location 对象的方法

assign() 加载新的文档
reload() 重新装载当前文档
replace() 装载一个新文档，无须为它创建一个新的历史记录
