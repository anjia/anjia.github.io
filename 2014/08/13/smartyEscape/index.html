探究下 smarty 对谁做了什么样的转义。

smarty 的转义

可对以下内容进行转义（转码）

  1. html 可转义 & “ ‘ < >
  2. htmlall 可转义 all html entities
  3. url 可转义 : / ? = & # 汉字
  4. mail 可转义 @ .
  5. quotes
  6. hex 十六进制
  7. hexentity 十六进制实体
  8. javascript

参考 smarty3 手册：http://www.php100.com/manual/smarty3/language.modifier.escape.html

双引号串 vs. 单引号串

在 php（smarty 同理）中，双引号串中的内容可以被解释被替换，而单引号串中的内容总被认为是普通字符。
